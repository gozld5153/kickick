# 서비스 소개 

<img src="https://cdn.discordapp.com/attachments/911193271609491487/921280363886686210/kickick-001.jpg" />

### 당신의 비법을 팔고, 타인의 비법을 사서 배우는 KICKICK 입니다.

- 킥머니라는 시스템으로 다른 사람의 킥을 배우고 자신의 킥을 팝니다.
- 다른 사람의 킥을 평가할 수 있습니다.
- 킥 구매 전 해당 킥에 대한 간략한 정보를 제공하여 구매자의 판단을 돕습니다.
- 킥 게시판 이외에 일반 게시판에서는 자유로이 글을 쓰고 지울 수 있습니다.
- 마이페이지에서는 자신의 모든 활동을 관리할 수 있습니다.
- 자신의 글에 댓글이 달리거나 공지가 등록되면 실시간으로 알람이 옵니다.
- 라이트모드와 다크모드를 제공합니다.

[**Kickick 바로가기**](https://kickick.net/)
